# Театр 'Мелодия' - Система управления театром 
 
## Описание проекта 
Консольное приложение для автоматизации работы театра. 
 
## Функциональность 
- Управление репертуаром театра 
- Продажа и бронирование билетов 
- Учет актеров и их ролей 
- Информация о залах и схемах мест 
- Отчетность и статистика
